# Calyx 🌿
Calyx is a web application for tracking calories and calculating BMI.  
The project is currently in active development.

## 🚧 Project Status
This project is still under development.  
New features and improvements are being added step by step.

## 🎯 Goal
The goal of Calyx is to provide a simple and minimalistic tool for:
- tracking daily calorie intake
- calculating BMI
- building awareness of eating habits

## 🧩 Planned Features
- Calorie tracking system
- BMI calculator
- Daily statistics
- Simple and clean UI
- Responsive design for mobile and desktop
